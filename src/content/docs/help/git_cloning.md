title: "Guide for cloning"
---

1.  Launch VS Code from Launcher (Mac) or Start Menu (PC)

2.  On the `Welcome` screen click `Clone Repository`:

<img src="/26SU-ASIAAM-191A/help/media/git_clone1.png" alt="">

1.  Click `Clone from GitHub`

<img src="/26SU-ASIAAM-191A/help/media/git_clone2.png" alt="">

4.  Allow GitHub to sign in Accept the following dialogue box:

<img src="/26SU-ASIAAM-191A/help/media/git_clone3.png" alt="">

1.  Click `Continue` in the web browser that opens up:

<img src="/26SU-ASIAAM-191A/help/media/git_clone4.png" alt="">

6.  Click `Choose Application`

<img src="/26SU-ASIAAM-191A/help/media/git_clone5.png" alt="">

7.  Click Open Link:

<img src="/26SU-ASIAAM-191A/help/media/git_clone6.png" alt="">

8.  Click `Open`:

<img src="/26SU-ASIAAM-191A/help/media/git_clone7.png" alt="">

1.  Search for the class repository by typing `26SU-ASIAAM-191A` and
    then select the repository: **albertkun/26SU-ASIAAM-191A**

<img src="/26SU-ASIAAM-191A/help/media/git_clone8.png" alt="">

10. Select a folder to clone into:

<img src="/26SU-ASIAAM-191A/help/media/git_clone9.png" alt="">

11. Click `Open` in your locally cloned repo:

<img src="/26SU-ASIAAM-191A/help/media/git_clone10.png" alt="">

12. You can see the files by clicking on the Explorer to the right:

<img src="/26SU-ASIAAM-191A/help/media/git_clone11.png" alt="">

13. Now you are ready for all labs!
