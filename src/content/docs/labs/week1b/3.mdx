The syntax for adding a marker in MapLibreGL is as follows:

```js
let longitude = -118.45;
let latitude = 34.05;

// Add a marker to the map
const marker = new maplibregl.Marker()
    .setLngLat([longitude, latitude])
    .addTo(map);
```

In this snippet, `longitude` and `latitude` are variables holding the coordinates of the marker. You can add multiple markers by repeating this pattern with different coordinates.

## Resources to find coordinates

- **Google Maps**: Right-click on any location and select "What's here?" to get coordinates. [Google Maps](https://www.google.com/maps)
- **OpenStreetMap**: Right-click on any location and select "Show address". [OpenStreetMap](https://www.openstreetmap.org)
- **Get Lat Long**: Enter an address to get coordinates. [GetLatLong](https://www.latlong.net/)

:::note[What is a geographic coordinate?]
A coordinate is a pair of numbers that specifies a location on Earth's surface. Coordinates are given in **latitude** and **longitude**, measured in degrees.

- **Latitude** specifies north-south position (like **altitude** goes up and down)
- **Longitude** specifies east-west position (like something **long** stretching left to right)

One easy memory trick: **Lat**itude sounds like l**a**dder — you go **up and down**. **Long**itude goes across — it's **long** like a horizontal line.
:::

:::caution[Reversed Coordinates!]
In MapLibreGL, coordinates are in `[lng, lat]` order — the **opposite** of the usual `[lat, lng]` order used in most other contexts (like Google Maps). Always double-check the order when adding markers!
:::

## Adding Popups

Popups appear when a user clicks a marker. Here's the basic syntax:

```js
const marker = new maplibregl.Marker()
    .setLngLat([-118.45, 34.05])
    .setPopup(new maplibregl.Popup({ offset: 25 })
        .setHTML('Hello! This is an HTML popup — I can use any <b>HTML tags</b> here!'))
    .addTo(map);
```

### Styling Popups with CSS

You can add a class to the popup to style it:

```js
const marker = new maplibregl.Marker()
    .setLngLat([-118.45, 34.05])
    .setPopup(new maplibregl.Popup({ offset: 25 })
        .setHTML('<div class="awesomePopup">Hello! This popup has a custom style.</div>'))
    .addTo(map);
```

Then in your `styles/style.css`:

```css
.awesomePopup {
    background-color: #f8f9fa;
    color: #212529;
    font-size: 16px;
    padding: 10px;
}
```
