Guide for installing Git SCM

  1. Go to https://git-scm.com/
  2. Download and install Git for your operating system using the steps below.

Follow the section for your operating system, then continue with After installing (all platforms).

Windows (PC)

  1. Run the installer and go through the installation wizard.

  2. Launch the command prompt when done by pressing the Windows key and typing cmd.
